Cloudflare DDNS 更新器

這是一個基於 Rust 的 Cloudflare DDNS 更新器，可以自動檢測和更新您的域名 IP 地址。

主要功能

  • 簡單易用的 Web 界面管理 DNS 記錄
  • 支持 IPv4 和 IPv6 記錄
  • 自動檢測公網 IP 變更
  • 定時更新 DNS 記錄
  • 多域名支持
  • 自訂更新間隔時間

開始使用

點擊下方按鈕進入 DDNS 管理界面，開始配置您的域名。

進入 DDNS 管理界面

使用說明

使用本工具，您需要：

  1. 一個 Cloudflare 帳號
  2. 在 Cloudflare 上管理的域名
  3. Cloudflare API Token（需要 DNS 編輯權限）

在 DDNS 管理界面中，您可以輸入 API Token，然後管理您的 DNS 記錄，並設置自動更新。
